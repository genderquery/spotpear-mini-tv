Example project for [Spotpear Mini TV].

* ESP32-C3 (Single core, Wi-Fi, Bluetooth)
* ST7735 128x128 16-bit color I2C TFT LCD
* 3 user buttons (IO8, IO9, IO10)
* 1 red LED (IO11)
* 1 reset button
* Micro USB port
* W25Q128 16MB SPI flash
* PL4054 Li-ion battery charger
* Headers for IO1, IO6, IO7, IO20, IO21, 3.3V, 5V

See [schematic].
